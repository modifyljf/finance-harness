AI Stock Review Video Harness — MVP Design

1. Goal

Build a minimal AI-powered harness that generates:

  • HTML animated PPT website
  • Stock review narration script
  • Structured slide JSON

from a single stock ticker input.

Final output should be directly usable for:

  • YouTube recording
  • OBS screen capture
  • TTS narration

2. MVP Scope

Included Not Included Yet
Stock analysis generation Automatic video rendering
HTML slide generation YouTube upload
Narration generation Multi-agent orchestration
CLI execution Cloud deployment

9. HTML Rendering Strategy

AI should NOT directly generate HTML.

Correct approach: 

AI → slides.json
Program → render HTML template

Benefits:

  • Stable output
  • Reusable templates
  • Easy styling
  • Easy animation upgrades
  • Deterministic rendering
