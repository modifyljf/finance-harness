## fish.audio 情绪标记规则

### 格式
- 标记必须放在句子**最开头**，格式：`(emotion) 句子内容`
- 一句话只能有一个情绪标记
- 错误示例：`股价在(excited)暴涨后回落` ❌
- 正确示例：`(excited) 股价单日暴涨百分之十四！` ✓

### 财经视频推荐标记库

| 场景 | 推荐标记 | 示例句型 |
|------|---------|---------|
| Hook / 开场钩子 | `(serious)` `(excited)` | 反常识陈述、悬念问句 |
| 数据揭示 / 超预期 | `(surprised)` `(confident)` | "实际EPS是..." / "数据显示..." |
| 看多逻辑展开 | `(excited)` `(optimistic)` | "第一点..." / "最强信号是..." |
| 看空风险警示 | `(worried)` `(serious)` | "但是..." / "风险在于..." |
| Why Now 核心叙事 | `(excited)` `(curious)` | "为什么是现在？" / "市场还没发现..." |
| 叙事缺口揭示 | `(sarcastic)` `(confident)` | "市场认为..." / "而现实是..." |
| 技术面分析 | `(calm)` `(confident)` | 支撑位/阻力位的陈述 |
| 风险提示段落 | `(serious)` `(nervous)` | "第X个风险..." / "一旦..." |
| 结论/总结 | `(determined)` `(calm)` | 投资建议、仓位管理 |
| 点赞引导 | `(happy)` | 最后的引导语 |
| 免责声明 | `(calm)` | 最后一句 |

### 停顿控制
- 段落之间自然停顿，不需要额外添加 `(break)`
- 如需强调某句话后的停顿，可在句末加 `(break)`

## 标注原则

1. **不要每句都打标记** — 每 3-5 句打一次，让情绪有节奏感
2. **情绪要与内容匹配** — 风险段落必须用 worried/serious，不能用 excited
3. **重要结论句必须标注** — 核心论断、目标价、Why Now 结论句必须有标记
4. **Hook 句必须标注** — 开场第一句必须有情绪标记
5. **情绪弧线** — 全文应有起伏：开场紧张 → 数据揭示兴奋 → 风险警示严肃 → 结论坚定
